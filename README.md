# Kursovaia

## О проекте

Проект состоит из двух частей:
- Frontend: статический интерфейс для просмотра матчей, аналитики и работы с пользовательскими данными.
- Backend: ASP.NET Core API, которое отдает данные матчей, пользователей и выполняет бизнес-логику.

## Frontend (папка front)

Frontend реализован на HTML, CSS и JavaScript без фреймворков.

Что делает фронт:
- показывает линию матчей и live-страницу;
- отображает статистику по матчам и калькулятор value;
- поддерживает регистрацию и вход пользователя;
- позволяет работать с профилем пользователя;
- отправляет запросы в backend API;
- запускает экспорт данных в Excel через endpoint backend.

Основные страницы:
- index.html: линия матчей;
- live.html: live-матчи;
- Statics.html: статистика и расчет value;
- Analiz.html: аналитика;
- profile.html: профиль пользователя.

## Backend (папка back)

Backend реализован на .NET 8 (ASP.NET Core Minimal API) с Entity Framework Core и SQL Server.

Что делает бэк:
- хранит и отдает матчи, команды и коэффициенты;
- выполняет регистрацию и вход пользователей;
- хранит избранные матчи пользователя;
- позволяет пользователю добавлять и редактировать свои матчи;
- рассчитывает value по коэффициенту и вероятности;
- получает xG-данные через внешний сервис;
- экспортирует данные в Excel (users + matches).

Технически используется:
- ASP.NET Core Minimal API;
- Entity Framework Core;
- SQL Server;
- CORS для связи с фронтом;
- HttpClient для внешних API;
- Hosted Service для фонового обновления коэффициентов.

## Запуск

Backend:

```bash
cd back/Kursovaia.Api
dotnet run
```

Frontend:
- открыть папку front через локальный HTTP-сервер (например, Live Server),
- затем открыть нужную страницу (например, Statics.html).

По умолчанию backend доступен на:
- http://localhost:5262
- http://localhost:5262/api/excel -- импорт excel
